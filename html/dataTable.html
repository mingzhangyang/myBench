If not checked, please provide a series of column names separated by comma, otherwise the default ones (e.g. Col #1) will be used. Of note, you can select only the columns you are interested and list them in the order of your own and ignore the others.

  • This is a demo page for data2table, a JavaScript framework to create HTML tables.
  • The left side bar contains all the control elements(icons) to manipulate the table.
    • Info: you see the words because you clicked it!  :-)
    • Local: click it to upload a local file (CSV, TSV, or JSON).
    • Paste: click to show the area to paste your content into.
    • Online: click to input an url to fetch online data.
    • Edit: click to edit the table, i.e. selecting columns to show/hide, reordering the columns, add/hide filter section, and more.
    • Graph: click to select columns of interest to visualize.
    • Style: click to select a styling of your favorite.
    • Export: click to export the processed (filtered, sorted, truncated) data in a format of interest.
  • The middle section is preview/edit area. Click the double arrows to fold it if you want to view the table in full page.
  • The right section (below) is table area, including graphs if it is set.
