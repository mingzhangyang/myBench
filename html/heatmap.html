Select your file to visualize.

Choose the color mode of heatmap.

Tips:

  • File format supported: .csv file
  • The first line should be column names, which then will be X axis labels
  • The first column should be sample names, which then will be the Y axis labels
  • The other content should be your numeric data
  • If you choose the color mode 2 above, you should process your data first, in which all the values are divided by the threshold value
