Requirements for using this App properly:

  • Note: This app create column-wise heat map.
  • File format supported: .csv file. If your data is in Excel, please export as CSV file.
  • The first row should be column names, which will be set as X axis labels.
  • The first column should be sample names, which will set as Y axis labels.
  • The other content should only be numeric values, otherwise the background will be fully dark.
  • For colors selection, please input the RGB values separated by comma.
  • Please refer to this color converter to find the rgb values of the color of your interest.
  • To use mode 2, it's better to provide an array of threshold values, one value for one column.
  • In color mode 2, the median value of each column will be set as threshold value of that column if the threshold values are not provided.
